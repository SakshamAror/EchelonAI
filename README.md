# EchelonAI — Cultural Signal Intelligence Platform

> **Not prediction. Not noise. Cultural + financial signal, unified.**

EchelonAI explains *why* a stock moved the way it did during a given quarter by combining real financial metrics, web-sourced cultural signals, and LLM synthesis — all grounded in data, never in speculation.

---

## Features

- **Equity Search** — Live Yahoo Finance autocomplete for any publicly listed stock
- **Quarterly Financial Metrics** — P/E, ROE, margins, FCF, debt/equity, growth rates and more via yfinance
- **Stock Price Chart** — Daily close prices for the selected quarter with peak annotation, start/end/high/low stats
- **Cultural Signals** — News articles from reputable outlets (Reuters, Bloomberg, CNBC, WSJ, FT, BBC, Guardian, AP News) fetched via Tavily, sentiment-scored and ranked by outlet traffic weighting
- **Echelon Score** — Composite signal score (0–100) blending financial fundamentals, cultural sentiment, and media momentum
- **Echelon Synthesis** — Past-tense retrospective analysis generated by Groq LLM (Llama 3.3 70B), grounded only in the fetched data with cited metrics and signals
- **Animated Loading Screen** — Real-time step progress showing each pipeline stage as it runs concurrently with data fetching
- **Metric Color Coding** — Significant financial metrics highlighted green/red only when thresholds are clearly crossed; neutral values stay uncolored
- **Demo Mode** — Three pre-built examples (Nike Q3 2024, Nvidia Q3 2024, Tesla Q4 2023) with live metric overlays

The Vite dev server exposes five middleware routes bridging the frontend to the data layer:

| Route | Handler | Purpose |
|---|---|---|

| `/yahoo-metrics` | `fetch-yfinance-metrics.mjs` | Live yfinance metrics |

| `/yahoo-resolve` | `resolve-yahoo-ticker.mjs` | Company name → ticker |
| `/alpha-synthesis` | Groq API (vite.config.ts) | LLM past-tense synthesis |

## Echelon Score

The Echelon Score (0–100) is a composite of three independently computed signals:

```
Echelon Score = 0.55 × Financial Score
              + 0.30 × Cultural Score
              + 0.15 × Forum Momentum
```

### Financial Score
Base score of 50, adjusted by clamped weighted contributions:

| Metric | Weight | Direction |
|---|---|---|
| Revenue Growth | 60× | Higher → better |
| Earnings Growth | 60× | Higher → better |
| Return on Equity | 45× | Higher → better |
| Return on Assets | 80× | Higher → better |
| Profit Margins | 45× | Higher → better |
| Debt / Equity | 6× inverted | Lower → better |
| Trailing P/E | 0.7× inverted | Lower → better |
| PEG Ratio | 5× inverted | Lower → better |
| Current Ratio | 6× (from 1.0) | Higher → better |

### Cultural Score
Sentiment-weighted score from reputable news articles, using outlet traffic as signal weight:

```
score = 78 + 40 × sentiment_index
           + 10 × polarity_ratio
           +  5 × impact_index
```

- **sentiment_index** — traffic-weighted average article sentiment (−1 to +1)
- **polarity_ratio** — (positive_weight − negative_weight) / total_weight
- **impact_index** — avg(relevance_scores) / 12, capped at 1.0
- Outlet weights are capped at 35% of total to prevent single-outlet dominance

Reputable outlets: Reuters, Bloomberg, CNBC, Financial Times, WSJ, AP News, BBC, The Guardian

### Forum Momentum (internal)
```
forum_momentum = 0.85 × cultural_score
               + min(12, article_count × 0.65 + avg_relevance × 0.5)
```

## Getting Started

### Prerequisites
- Node.js 20+
- Python 3.12+
- API keys: `TAVILY_API_KEY`, `GROQ_API_KEY`

### Install

```bash
# Clone and install frontend dependencies
git clone https://github.com/your-org/AIStockScreener.git
cd AIStockScreener/frontend
npm install

# Create Python virtual environment and install dependencies
python3 -m venv .venv
.venv/bin/pip install yfinance python-dotenv tavily-python
```

### Environment Variables

Create `frontend/.env`:

```env
TAVILY_API_KEY=your_tavily_api_key
GROQ_API_KEY=your_groq_api_key

# Optional overrides
GROQ_MODEL=llama-3.3-70b-versatile
AGENT_PYTHON_BIN=.venv/bin/python
```

### Run

```bash
cd frontend
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
